# QRCode Terminal

QRCodes, in the terminal

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     qrcode_terminal:
       github: mamantoha/qrcode_terminal
   ```

2. Run `shards install`
